Белые Списки: Теория и Обход

Белые Списки (White Lists) — это механизм, с помощью которого разрешается доступ только к определённым ресурсам.
Чем они отличаются от Чёрных списков?
Характеристика Белый список (Whitelist) Черный список (Blacklist)
Основной принцип Запрещено всё, что не разрешено явно. Разрешено всё, что не запрещено явно.
Логика проверки Система ищет объект в списке разрешенных. Система ищет объект в списке запрещенных.
Действие по умолчанию Блокировка (Default Deny). Пропуск (Default Allow).
Если объекта нет в списке Доступ отклоняется. Доступ предоставляется.

2. Список Приложений

  • 1. IOS - Streisand, Karing
  • 2. Windows - v2rayN, Throne, Karing, singbox-launcher
  • 3. MacOS - v2rayN, Throne, Karing, singbox-launcher
  • 4. GNU/Linux - v2rayN, Throne, Karing
  • 5. Android - NekoBox, v2rayNG, Karing

3. Добавление подписок

Throne: Приложение > Добавить профиль из буфера обмена > Создать новую группу подписки
Karing: Добавить профиль > Импорт из буфера обмена > Примечание любое > Выбираем сервер
v2rayN: Серверы > Импорт массива URL > Группа подписки > Обновить подписку без прокси
NekoBox: Три полоски > Группы > Плюсик > Тип: Подписка > Вставляем ссылку
Streisand: Плюсик > Добавить вручную > Тип: Subscription > URL вставляем ссылку
singbox-launcher: Wizard > в Subscription URL вставляем ссылку > Save > Start > Server > Выбираем сервер

4. Использование и логика запуска

1
После добавления подписки в клиент запускаем URL Тест и сортируем список по задержке.
2
Перебираем конфиги с самой маленькой задержкой. Какой будет работать — тот и используем.
3
Если не отвечает ни один конфиг — обновите подписку.
⚠️
Важное примечание: Для обновления подписки необходим нормально работающий Wi-Fi при включённых «Белых Списках».

5. Понятия и дополнительная настройка

DoH (DNS over HTTPS) / DoT (DNS over TLS): Шифрование DNS-запросов от слежки.

Stack (TUN): Стек сетевого интерфейса. Рекомендуется gVisor.

MTU: Размер пакета. Для Hysteria2 критично значение 1500.

Рекомендуемые параметры:

  • Удаленный DNS: https://1.1.1.1/dns-query
  • Stack (TUN): gVisor
  • MTU: 1500
